# Community App

A full-stack community platform where users can create an account, log in, and share stories in a supportive environment.

## Features

- User registration
- User login
- Create stories
- View community stories
- Edit stories
- Delete stories
- Responsive frontend
- REST API with Flask
- PostgreSQL database

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Python
- Flask

### Database
- PostgreSQL
